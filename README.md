# NordFlow

NordFlow är ett personligt lärandeprojekt där jag bygger ett enklare ERP-system från grunden till slut.

Syftet med projektet är att bli bättre på programmering, struktur, problemlösning och att arbeta med ett större system över tid.

## Mål
- Bygga ett enklare ERP med flera delar och funktioner
- Träna på att planera, utveckla och förbättra en applikation steg för steg
- Få bättre förståelse för hur ett större system hänger ihop

## Planerade funktioner
- Hantering av kunder
- Hantering av produkter
- Ordrar
- Lagerstatus
- Enkel orderöversikt

## Teknik
- C#
- .NET
- Rider

## Status
Projektet är under utveckling och används främst för lärande.
